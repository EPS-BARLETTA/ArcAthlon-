Arcathlon — Conseils CENTRAUX (remplacent les anciens boutons/options)
======================================================================

Ce pack modifie uniquement la zone de tir :
- Les conseils apparaissent **au centre**, à la place des anciens boutons d’options/validation.
- Affichage **après chaque flèche** et **en fin de course** (les deux).
- Le message **reste affiché** jusqu’au prochain événement (nouvelle flèche ou fin).

Tout le reste est inchangé : 2 coureurs, chrono, règles par blocs, résumé, QR/CSV, mode Prof.

Intégration : remplace ton `index.html` par celui de ce pack.
