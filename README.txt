Arcathlon — Version avec page d’accueil + CONSEILS persistants
=================================================================

Ce paquet est basé sur ta version (accueil + saisie duo + course + bilan QR/CSV) auquel j’ai **réintégré les aides** :
- **Conseils persistants** : un conseil s’affiche et **reste** jusqu’à la prochaine flèche ou la **fin**.
- **Options simples** : `Activer les conseils` / `Après chaque flèche` / `En fin de course`.
- **Génération de conseils** basée sur la **distribution des scores** et l’**indice arc**.

Rien d’autre n’a été modifié : chrono, tours/blocs flèches, résumé, QR/CSV, **mode Prof** restent identiques.

➡ Utilisation
1) Ouvre `index.html` dans ton navigateur.
2) Active/désactive les options de conseil dans le panneau **Tir à l’arc**.
3) Les conseils **ne disparaissent pas** tout seuls : ils restent en place jusqu’au prochain événement.

➡ Déploiement GitHub
- Remplace le `index.html` de ton dépôt par celui-ci, commit + push.
